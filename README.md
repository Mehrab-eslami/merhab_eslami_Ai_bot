# ChatGPT Telegram Bot + Admin Panel

✅ ربات تلگرام با پاسخ هوش مصنوعی (GPT 3.5 via Aivix API)  
✅ ذخیره پیام‌ها در دیتابیس  
✅ پنل مدیریت پیام‌ها (تحت وب)

## اجرای ربات
```bash
pip install -r requirements.txt
python bot.py
```

## اجرای پنل مدیریت
```bash
python panel.py
```

📍 پنل در `http://127.0.0.1:5000`
